# 喵机妙算 Meowracle Machine

一个给真实猫滚键盘用的猫神谕解释器。

## 运行

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

## 演示流程

1. 打开页面后，把光标留在中央大输入框里。
2. 邀请猫自由路过键盘、踩一下、蹭一下、尾巴扫一下。
3. 猫滚完后，由人类按 `Command + Enter` 或 `Ctrl + Enter` 提交。
4. 右侧会生成每日猫谕卡，包含原文、猫神解读、关键词、建议、幸运物、禁忌和误读警告。

没有猫时可以点「模拟猫神」兜底。

## 当前实现

- 静态单页 App，无需安装依赖。
- 本地猫谕生成逻辑，不依赖后端或 API。
- 支持复制分享文本。
- 支持下载 PNG 猫谕卡。
- 支持多种猫神人格。
