# Rolling Cat

一个给真实猫滚键盘用的产品灵感生成器。

猫的键盘输入不会被解释成运势，而是作为随机种子去撞词库。每个爪印会映射到一个词条，原型优先使用 Wikipedia 随机摘要；随后产品破解层会把这些互不相关的词条解释成一个可以 vibe coding 的产品方向。

## 运行

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

## 演示流程

1. 选择今天想要的产品灵感类型。
2. 点击「开始接收」，进入猫键盘安全模式。
3. 邀请猫自由路过键盘、踩一下、蹭一下、尾巴扫一下。
4. 键盘事件会被拦截，只作为随机种子记录，不会触发刷新、跳转或提交。
5. 点击「猫滚完了，生成产品思路」。
6. 页面会抽取词条，并生成产品 brief、目标用户、核心机制、MVP 和传播钩子。

没有猫时可以点「没有猫，我伪装一下」兜底。

## 当前实现

- 静态单页 App，无需安装依赖。
- 单栏极简交互，参考隔壁 Rolling Cats 的降临台流程。
- 猫键盘安全模式会拦截键盘事件。
- 词条优先来自 Wikipedia 随机摘要，网络不可用时使用本地 fallback 词库。
- 可选 AI 服务接口：如果提供 `POST /api/idea`，页面会用服务返回的 idea 覆盖本地产品 brief。
- 支持复制生成出的产品 brief。
