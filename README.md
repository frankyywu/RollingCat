# 喵机妙算 Meowracle Machine

一个给真实猫滚键盘用的猫神谕解释器。

## 运行

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

## 演示流程

1. 打开页面后，选择今日值班猫神人格。
2. 人类点击「开始接收」，进入猫键盘安全模式。
3. 邀请猫自由路过键盘、踩一下、蹭一下、尾巴扫一下；所有键盘输入都会被拦截，只记录成猫语。
4. 猫滚完后，由人类点击「收谕」提交。
5. 右侧会生成每日猫谕卡，包含原文、猫神解读、关键词、建议、幸运物、禁忌和误读警告。

没有猫时可以点「模拟猫神」兜底。

## 当前实现

- 静态单页 App，无需安装依赖。
- 本地猫谕生成逻辑，不依赖后端或 API。
- 猫键盘安全模式会拦截键盘事件，避免猫误触提交、刷新、跳转或按钮。
- 支持复制分享文本。
- 支持下载 PNG 猫谕卡。
- 支持多种猫神人格。
